# Contributing

This is a 5-person capstone project. Every team member contributes via a **fork &
pull request**, editing only their own component and filling in their individual
contribution doc.

👉 **Read the full step-by-step in [`docs/TEAM_GUIDE.md`](docs/TEAM_GUIDE.md).**

Quick version:
1. **Fork** this repo to your own GitHub account.
2. Clone your fork and set `git config user.email` to your GitHub email (so the
   commit is attributed to you).
3. Edit **only your own files** (see the ownership table in the team guide) and make
   one real improvement.
4. Rename + fill in your doc in `docs/individual_contributions/`
   (`Mx_role_<yourname>.md`, replace `<Member N name>` with your name).
5. Run `python -m eval.run_eval` (must stay green).
6. Push to your fork and open a **pull request** into
   `shivam1234100/Contact-Guard:main`.

After your PR is merged you'll appear under the repo's **Insights → Contributors**.
